# comment-analyzer
данный веб-сервис создан в рамках итоговой работы по предметной области машинное обучение. Модель анализирует тон текста и предсказывает к какому классу относится предложение(негативный,позитивный,нейтральный). В репозитории находится сама модель, веб-сервис созданный с помощью библеотеки "gradio", и все файлы необходимые для корректной работы модели.
